Project 2: Visual Analytics — Amazon Product Engagement

Dataset: ikramshah512 — Amazon Products Sales Dataset (42k items, 2025).

Problem

What drives engagement on Amazon? Specifically, I explored: (1) how reviews are distributed across products, (2) whether ratings actually distinguish success, (3) which categories dominate, and (4) how discounts affect reviews and satisfaction.

Data

CSV with columns: discounted_price, original_price, discount_percentage, product_rating, total_reviews, product_category. ~42,000 products scraped from Amazon. Data is not official but is useful for practicing visual analytics and pattern-finding.

Pre-processing

  • Converted numeric fields (price, discount, rating, reviews) to floats/integers.
  • Dropped rows missing critical values.
  • Clipped extreme outliers in reviews using log scale to handle heavy skew.
  • Created discount_band categories (No Discount, 0–20%, 20–50%, 50%+).
